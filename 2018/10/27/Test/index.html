github 搭建个人博客

  1. 下载并安装 node.js 按照安装程序执行最后会自动添加环境变量path

  2. 打开cmd

  3. hexo（$ npm install -g hexo-cli）

  4. hexo init 你的博客文件夹
    cd 你的博客文件夹(确保在博客文件夹)

    npm install
hexo  new  test.md  
hexo  s    (hexo server 全命令)
hexo  g    (hexo generate 全命令)

  5. 下载并安装 Git

  6. github 添加公钥

  7. 配置站点信息的-config.yml
