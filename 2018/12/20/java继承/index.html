java 继承

// 所有类的祖先类

Father类继承Grandpa类重写fun方法

Son类继承Father类重写fun方法

Sun类继承Son类重写fun方法

测试类

关于java继承的说明：

构造方法的执行从父类开始依次向上找，直到找到最开始的祖先类，然后祖先类依次向下执行直到执行完所有的构造函数

继承是子类存有父类引用，并通过此引用来寻找父类，而不是直接在父类上进行扩充，当子类复写父类方法时，会通过引用依次向上找，找到父类方法将其复写，这样即使将子类对象强转成父类对象其方法也是最后一次复写的，而不一定是父类中的方法（例中的fun方法，Grandpa定义而被每一个子类重写，每一个子类都通过自己的Grandpa引用找到fun方法并将其重写，对于grandpa对象来说fun方法没有改变，

对于father对象来说fun方法

对于son对象来说fun方法

对于sun对象来说fun方法

继承是引用的传递并不是类的扩展
